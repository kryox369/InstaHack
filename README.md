# InstaHack
Un script hecho en Python para hacer ataques de Fuerza Bruta a cuentas de Instagram

----INSTALACION---

    git clone https://github.com/kryox369/InstaHack.git
    pip3 install colorama
    cd InstaHack
    python3 InstaHack.py
